==================================================
🎯 MASTER GOALS (What we are building)
==================================================
1. A NATIVE agentic browser platform — NOT just wrapping Chrome externally
2. The physical browser must be EMBEDDED INSIDE the Nexus tool UI, visible and interactive
3. Multi-Agent system: Planner → Automation → Memory → Verifier agents working in concert
4. Persistent session/cookie handling so agents don't need to re-login every time
5. Capable of accepting big complex tasks, autonomously breaking them into steps, executing,
   verifying results, and replanning if needed.
6. Graceful lifecycle: when the Nexus UI closes, all spawned browsers/processes die with it

🚧 ACTIVE OBSTACLES: Contextual Verification & Multi-Tab Stability
==================================================
SYMPTOM: The engine is now fast and handles auth correctly. However, complex tasks spanning multiple tabs or deeply nested iFrames still require more robust context tracking.

ROOT CAUSES:
- Playwright context for 'get_current_page' might lag if the agent spawns new windows.
- The Verifier agent needs more 'eyes' on the DOM to detect when a goal is TRULY finished.

FIX STRATEGY: 
- Implement a DOM-Tree summarizer to feed the Verifier.
